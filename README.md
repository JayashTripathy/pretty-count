# Pretty Count

> Javascript library to format numbers in a pretty human readable format

## Installation

Install Package

```shell
npm i pretty-count
```

Import utitlity

```js
import pc from 'pretty-count';
``` 

Use the utility

```js
pc(1000000) // 1million
pc(1000000000, {
    scale: "indian",
    showFullSymbol: false
}) //1Ar
```
